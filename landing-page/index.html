About Us

Our dream was to make an IOT project that combined hardware and software in a useful, yet inexpensive and escalable way. With PAVApp, the goal is to let the world create one, improving, customizing, and making it their own.

1 Raspberry Pi

Server running VUE.js webpage that makes requests to the device. Runs on Raspbian OS, automatically runs the web page service on boot.

2 Device

This device contains a SONOFF Mini with an ESP8266 chip for WiFi connection, a relay and a temperature sensor. This device takes the requests and runs functions to turn on/off the relay and returns the state of it in conjunction with the current temperature.
